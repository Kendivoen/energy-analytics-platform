# Energy Analytics Platform

Integrated renewable energy analytics project combining:

- Electricity demand forecasting
- Solar generation prediction
- Carbon impact reporting

## Objectives

1. Forecast electricity demand
2. Predict solar PV generation
3. Quantify carbon emissions avoided
4. Develop professional analytics dashboards

## Tools

- Python
- Pandas
- Scikit-learn
- XGBoost
- Jupyter
- Power BI
- Git/GitHub

Project status: In development
